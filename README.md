# Trip Planner

Минималистичный планировщик путешествий в стиле Apple Human Interface Guidelines.

## Возможности

### Список поездок
- Прошлые и предстоящие поездки
- Обложки с фото или автоматическими эмодзи
- Быстрый доступ к деталям

### Расписание
- События по дням
- Время и заметки к каждому событию
- Группировка по датам

### Ссылки
- Google Maps / Яндекс Карты
- Instagram места
- Сайты отелей, ресторанов, достопримечательностей
- Группировка по типу

### Заметки
- Текстовые заметки о поездке
- Важная информация, контакты, tips

### Фото
- Галерея фотографий
- Хранение в базе браузера (IndexedDB)

### Бэкап и шаринг
- Экспорт одной поездки в JSON
- Экспорт всех данных
- Импорт JSON (передача между участниками)

## Установка

### GitHub Pages
1. Создайте репозиторий на GitHub
2. Загрузите файлы: `index.html`, `manifest.json`, `sw.js`
3. Включите GitHub Pages в настройках репозитория
4. Откройте на телефоне и добавьте на домашний экран

### Локально
Просто откройте `index.html` в браузере.

## Использование на телефоне

### iOS (Safari)
1. Откройте страницу в Safari
2. Нажмите "Поделиться" (квадрат со стрелкой)
3. Выберите "На экран Домой"
4. Приложение будет работать как нативное

### Android (Chrome)
1. Откройте страницу в Chrome
2. Нажмите меню (три точки)
3. Выберите "Добавить на главный экран"

## Шаринг поездки

Чтобы поделиться поездкой с другими участниками:

1. Откройте поездку
2. Нажмите "+" -> "Экспорт поездки"
3. Отправьте JSON-файл участникам
4. Участники нажимают "+" -> "Импорт JSON" на главном экране

## Технологии

- Vanilla JavaScript (без фреймворков)
- IndexedDB для хранения данных
- Service Worker для офлайн-работы
- PWA для установки как приложение

## Дизайн

Интерфейс вдохновлен Apple Human Interface Guidelines:
- Минималистичный дизайн
- Тонкий заголовок сверху
- FAB-кнопка (плюсик) справа снизу
- Модальные окна снизу
- Blur-эффекты
- SF Pro шрифты
